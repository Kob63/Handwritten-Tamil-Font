FONTLOG for MyHands

This file provides information about the MyHands font.

Font Family Name: MyHands
Designer: Oldman
License: SIL Open Font License, Version 1.1
License URL: https://openfontlicense.org

Description:
MyHands is a personal handwritten Tamil font inspired by the natural
penmanship of an old farmer. Created entirely using a mouse, it captures
a rustic, human feel. The font is suitable for casual writing,
storytelling, education, and cultural or community projects.
